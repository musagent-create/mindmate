# MindMate 🧠

AI-samtalepartner for ældre med demens.

## Hvad er det?

En voice-first AI companion der kører på iPad/tablet, designet til ældre med demens. Den husker brugerens historier, relationer og interesser — og fører en varm, tålmodig samtale.

## Tech Stack

- **Frontend:** Next.js (React) → Vercel
- **LLM:** Claude API (Anthropic)
- **Voice → Text:** Whisper API
- **Text → Voice:** ElevenLabs
- **Database:** Supabase (PostgreSQL + pgvector)
